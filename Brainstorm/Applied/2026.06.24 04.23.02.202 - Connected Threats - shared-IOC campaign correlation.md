# Connected Threats - shared-IOC campaign correlation

Cost: M

Persist each file's sandbox IOCs (DNS hostnames + IPs from VtBehaviour.Network, dropped files, registry keys) into HashCacheEntry and build an in-memory inverted index IOC->files. In the detail pane show 'Shares N indicators with M other scanned files', list overlapping domains/IPs, link to peers, color red if any peer was malicious. Catches the low-score malicious sibling that phones the same C2 as a known-bad file. Reuses FetchBehaviourAsync; no new API call.
