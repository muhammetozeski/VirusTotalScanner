# Major-engine consensus split

Split per-engine detections into a small user-editable set of MAJOR engines (Microsoft, Kaspersky, ESET-NOD32, BitDefender, GData, Avast, Sophos, Malwarebytes, McAfee, TrendMicro, Symantec, Google) vs the rest. One-line trust strip in the detail pane: 'Major: 0 flagged - Minor: 3' (green if no major flags, red if any). Store the small summary (major-flagger names + count pair) in cache. Major set editable in settings. Zero extra API calls. Directly addresses the common false-positive case (a few obscure engines flag a clean installer). Example: ski32.exe flagged by 6 minor engines = false positive.

_From the autonomous brainstorm (10 groups, 2026.06.23 23.2x). Deduped from 5 near-identical 'find all copies' proposals + 5 distinct ideas._
