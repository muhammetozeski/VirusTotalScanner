# Live Downloads watch - auto-scan new executables

A background FileSystemWatcher on user-chosen folders (default Downloads/Desktop/%TEMP%) for newly created/renamed exec-class files. Debounce until the handle releases + size stops growing, then feed into the existing pipeline (trust pre-filter -> cache -> keyless GUI for unknowns only). Clean=silent; flagged=tray toast + click-through to detail, reusing verdict coloring + quarantine gate. Per-folder toggle + watched list in settings; session counter. Event-driven (catches the seconds between landing and double-click), unlike the timer sweep. Zero quota steady-state.
