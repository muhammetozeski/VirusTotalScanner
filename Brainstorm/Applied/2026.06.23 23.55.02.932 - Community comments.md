# Community comments

Show VirusTotal community comments in the detail pane (votes are already done). Capture the /ui/files/<hash>/comments response in the WebView2 GUI engine the same way the file report is captured, parse the top few comments (author, date, text, tags), and show them in a small read-only list under the votes line. Toggleable with the existing ShowCommunityVotes setting (or a sibling ShowCommunityComments).
