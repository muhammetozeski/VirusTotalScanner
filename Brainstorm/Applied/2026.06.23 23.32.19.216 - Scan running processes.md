# Scan running processes

A 'Scan what's running now' action (GUI button + CLI --running): enumerate live processes, collect+dedupe each on-disk image path (Process.MainModule.FileName, skip access-denied as 'unreadable'), feed the path set into the existing pipeline (trust pre-filter -> cache -> VT for unknowns only). Results render in the normal queue + folder rollup. Optional 'include loaded DLLs' toggle, off by default.

_From the autonomous brainstorm (10 groups, 2026.06.23 23.2x). Deduped from 5 near-identical 'find all copies' proposals + 5 distinct ideas._
