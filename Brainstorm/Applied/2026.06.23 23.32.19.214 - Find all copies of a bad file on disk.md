# Find all copies of a bad file on disk

On a malicious verdict, a 'Find other copies on this PC' action. Two-stage funnel: query the local Everything CLI (es, already on PATH) for candidates matching the bad file's exact byte size and/or name (fast), fall back to a scoped walk of common drop folders if es is absent; then hash only the candidates and keep exact SHA-256 matches. List paths, one-click 'quarantine all' through the existing .VIRUS confirm gate. Zero VT calls (reuses the known verdict).

_From the autonomous brainstorm (10 groups, 2026.06.23 23.2x). Deduped from 5 near-identical 'find all copies' proposals + 5 distinct ideas._
