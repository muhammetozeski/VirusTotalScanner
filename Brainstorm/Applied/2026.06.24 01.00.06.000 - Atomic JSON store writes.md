# Atomic JSON store writes

Rank: 06 (round-3 kalite/bug brainstorm, 24->23->10, satir-numarali dogrulanmis)

Robustness (M). Hicbir JSON store atomik yazmiyor: cache.json, history.json, quarantine manifest.json, scan-session.json + BaselineStore/IocStore/LedgerService/ProductSignerRegistry/ConfirmGate hepsi duz File.WriteAllText(finalPath). Crash/kill/guc-kesintisi yazma ortasinda truncated dosya birakir. Fix: tek paylasilan helper path+'.tmp' yazip File.Replace/Move(overwrite) ile gercek yola tasisin; her store'un save'i buradan gecsin. En kotusu quarantine manifest = .VIRUS dosyalarina tek index.
