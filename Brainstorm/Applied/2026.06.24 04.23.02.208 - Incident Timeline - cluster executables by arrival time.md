# Incident Timeline - cluster executables by arrival time

Cost: M

Machine-wide chronological triage: enumerate executables via Everything es, derive an arrival timestamp = earliest of FileInfo.CreationTimeUtc and the Zone.Identifier download mark, surface HostUrl. Bucket into day-level clusters (label as best-effort). Attach a verdict per file from the hash cache (zero quota) falling back to keyless GUI + trust pre-filter. Render a timeline grouped by day, expandable to files sorted by detection count with download host + verdict. User picks an incident window ('around June 10') and sees every executable that landed then ranked by risk. CLI --timeline --since/--until. New axis: time-of-arrival, distinct from baseline/persistence.
