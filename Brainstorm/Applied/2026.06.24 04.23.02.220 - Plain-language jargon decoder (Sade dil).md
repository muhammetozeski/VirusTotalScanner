# Plain-language jargon decoder (Sade dil)

Cost: S

A keyless 'Bunu acikla / Sade dil' layer on the detail pane decoding AV jargon where a newcomer meets it. A bundled static glossary (new JargonGlossary.cs, pure lookup): (1) VT engine categories (malicious/suspicious/harmless/undetected/timeout/type-unsupported/failure) -> one-line plain Turkish; (2) common detection-name morphemes (Heur/Gen/Generic, Trojan, PUA/PUP, Riskware, not-a-virus, Adware, Win32/W32, .MSIL) -> plain meanings. Wire via the existing Engines_CellFormatting hook: set DataGridViewCell ToolTipText on Kategori and Sonuc cells; a 'Sade dil' checkbox next to 'Tum motorlari goster' appends a gloss after each stats term. Onboarding/accessibility, zero quota.

_From round-3 autonomous brainstorm (10 proposer-vs-critic groups, 2026.06.24 04.2x)._
