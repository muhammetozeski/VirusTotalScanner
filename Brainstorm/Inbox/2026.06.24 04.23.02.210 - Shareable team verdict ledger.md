# Shareable team verdict ledger

Cost: M

Portable mergeable ledger built on cache.json. Export: ledger.json with per-hash {sha256, md5, verdict, detections/total, reportUrl, firstSeen, vettedByMachine, vettedAtUtc, note} + an integrity hash over rows. CLI --export-ledger (filter clean/malicious); GUI button. Import/merge: --import-ledger seeds the local cache for unknown hashes; on a known hash it records a conflict instead of overwriting, provenance-preserving. Lookup short-circuit: consult merged ledger before any VT/GUI call (like cache TryGet). --ledger-diff <theirs>: what they vetted that you haven't + verdict conflicts. No server/account/network — a file dropped in a shared folder/git. Compounds the cache's quota-saving across a team.

_From round-3 autonomous brainstorm (10 proposer-vs-critic groups, 2026.06.24 04.2x)._
