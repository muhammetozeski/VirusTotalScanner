# ItemFinished off-UI-thread crash

Rank: 01 (round-3 kalite/bug brainstorm, 24->23->10, satir-numarali dogrulanmis)

BUG (S). ScanScheduler.ProcessAsync finally ItemFinished?.Invoke'u Parallel worker thread'inde DOGRUDAN cagiriyor (sat 239) ama ProgressChanged UiPost'tan geciyor (390). Subscriber ScanQueueControl.OnItemFinished SelectedItem/_detail.Show/_view BindingList mutasyonu/_grid.DataSource'u arka-plan thread'inden, MaxConcurrency>1'de coklu worker'dan es zamanli yapiyor. Cross-thread DataGridView = aralikli InvalidOperationException/crash. Fix: ItemFinished?.Invoke'u finally icinde UiPost ile sar (ProgressChanged gibi). CLI etkilenmez (UiPost=identity).
