# Verdict Drift Report - shareable what-flipped artifact

Build a shareable artifact from the re-check sweep + cache (not a fresh scan). Capture a before-snapshot of HashCache (CachedUtc/Detections/TotalEngines/ReportUrl/votes/LastPath) and an after-snapshot, diff to find every file whose VT standing CHANGED: clean->malicious flips, detection-count deltas, newly-detecting major engines, vote shifts. Render via ReportWriter (reuse HTML; ADD a CSV format). Each row shows LastPath + old/new ratio + VT permalink. GUI 'Export drift report' button on the sweep dialog + CLI --drift-report <path> (extension picks html/json/csv). Zero extra quota (diff+render are local over two cache snapshots). Answers 'what changed', the most actionable local signal.

_From round-2 autonomous brainstorm (10 proposer-vs-critic groups, 2026.06.24 03.1x)._
