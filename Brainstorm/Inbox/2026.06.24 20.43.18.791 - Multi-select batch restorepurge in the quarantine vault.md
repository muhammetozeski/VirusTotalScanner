# Multi-select batch restore/purge in the quarantine vault

Kategori: ux | Efor: S | Rank: 10

## Ne
Set _grid.MultiSelect=true in QuarantineVaultDialog.cs and rewrite RestoreSelectedAsync / PurgeSelected to iterate _grid.SelectedRows instead of the single CurrentRow via Selected(), with one ConfirmGates.Quarantine.Ask for the whole batch and an aggregate result line ('7/9 geri yuklendi'). Add a 'Tumunu kalici sil' button that purges every listed entry after a single confirm.

## Neden
The scan queue already supports multi-select batch quarantine, but the vault that holds the results is single-row only (Selected() reads CurrentRow, every op pops one modal). After a copy-finder sweep quarantines a directory's worth of identical copies, clearing or restoring them means one dialog per file. Batch ops here match the queue's existing ergonomics and make post-sweep cleanup practical.
