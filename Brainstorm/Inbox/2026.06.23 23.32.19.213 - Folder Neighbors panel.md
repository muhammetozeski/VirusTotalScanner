# Folder Neighbors panel

On a malicious/suspicious verdict, show a Neighbors panel for the file's directory built from cache: other files in that folder already scanned (with stored verdict + first-seen), a count of never-scanned files present, and a one-click 'scan the rest now' for the un-scanned/stale siblings. Needs cache to also store last-seen path(s) per hash. Pure local reuse of cache.json + one Directory.EnumerateFiles.

_From the autonomous brainstorm (10 groups, 2026.06.23 23.2x). Deduped from 5 near-identical 'find all copies' proposals + 5 distinct ideas._
