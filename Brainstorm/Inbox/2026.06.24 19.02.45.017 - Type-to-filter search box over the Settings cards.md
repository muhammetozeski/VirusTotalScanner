# Type-to-filter search box over the Settings cards

Kategori: ux | Efor: M | Rank: 4

## Ne
Add a TextBox docked above _flow in SettingsControl.cs that filters which of the ~10 cards stay visible as the user types. Build a searchable string per card from its Card(title,...) title plus the Text of its child controls (CheckBoxes, labels, numeric-row labels), then set each card's Visible by query match. Esc clears and restores all cards.

## Neden
Finding 'otomatik karantina esigi' or 'onbellek gecerlilik' today means eyeball-scrolling 10 dense cards (~40 options). A live filter is the single biggest discoverability win for a settings screen this size, and nothing currently lets you jump to a setting by name.
