# Plain-language What should I do action card

A per-file decision card above the detail/engine table when a scan completes. A deterministic RecommendationService synthesizes signals already in memory (verdict+count, Family/ThreatLabel, trust status, Zone.Identifier origin, rarity/first-seen, folder-neighbors) into ONE of: 'Safe to keep' / 'Be cautious' / 'Remove now', plus one rationale sentence citing the strongest signal, plus 2-3 inline buttons reusing existing flows (Quarantine / Find copies / Keep). No new lookups. Localized via Strings/LocManager. Usability synthesis layer for non-experts.

_From round-2 autonomous brainstorm (10 proposer-vs-critic groups, 2026.06.24 03.1x)._
