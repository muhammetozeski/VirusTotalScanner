# Persistence hunt - autostart hooks of a detected file

On a malicious verdict, enumerate Windows persistence locations (Run/RunOnce HKCU+HKLM incl WOW6432Node, Startup folders, Scheduled Tasks via schtasks /query /xml, Services ImagePath) and list every entry whose command references the detected file by full path, bare name, or any copy from CopyFinderService. Read-only first; per-entry neutralize behind ConfirmGate with a reversible restore artifact (.reg export / task XML / moved shortcut) saved near the quarantine folder. Closes the IR gap: quarantining the binary leaves dangling autostarts that re-fetch the payload.
