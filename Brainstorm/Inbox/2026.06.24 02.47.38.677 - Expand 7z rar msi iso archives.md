# Expand 7z/rar/msi/iso archives

ZIP-family expansion is done (built-in ZipArchive). Extend ArchiveExpander to 7z/rar/tar/gz (SharpCompress NuGet, pure-managed) and msi/iso (OLE/ISO9660 readers, or shell out to 7-Zip from scoop). Keep the bounded extract-to-temp + cleanup flow; if a format cannot be opened, scan the archive file itself.
