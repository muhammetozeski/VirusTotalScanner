# Make the lifetime count tiles clickable, drilling into a pre-filtered History tab

Kategori: ux | Efor: S | Rank: 2

## Ne
Give each Tile in BuildTiles/Tile (ScanOverviewControl lines 112-137) Cursor.Hand and a Click handler raising a new GoToHistoryFiltered(category) event. MainForm switches to the Gecmis tab and applies the matching filter on ScanHistoryControl. 'Tehdit' reuses the existing _threatsOnly checkbox; 'Supheli' (detections>0 but below threat) and 'Temiz' (detections==0) need a small filter selector added to ScanHistoryControl.Reload, which today only has _threatsOnly/_starredOnly.

## Neden
The tiles show numbers that lead nowhere — a user who sees '5 Supheli' has no path from that number to the actual files (the shipped banner only drills into live threats, not Supheli/Temiz). Click-the-number-to-land-on-those-files turns the overview from a static dashboard into a navigable start.
